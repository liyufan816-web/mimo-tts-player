# MiMo-V2-TTS 语音播放器

基于小米 MiMo-V2-TTS 模型的文本转语音工具，支持多种语音风格。

## ✨ 功能特点

- 支持东北话、粤语、台湾腔等多种方言
- 支持开心、温柔、悄悄话等情绪控制
- 自动保存为 MP3 文件
- 调用系统播放器自动播放
